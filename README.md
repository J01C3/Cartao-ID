# 📱 Cartão Digital de Identificação
![Ionic](https://img.shields.io/badge/Ionic-Framework-blue?logo=ionic)
![Angular](https://img.shields.io/badge/Angular-Framework-red?logo=angular)
![Status](https://img.shields.io/badge/status-em%20desenvolvimento-yellow)

Projeto desenvolvido com **Ionic + Angular** para criar um cartão digital de identificação interativo, ideal para eventos acadêmicos como recepção de calouros.

## 🚀 Funcionalidades

✔️ Exibição de informações básicas  
✔️ Página "Sobre mim" com detalhes adicionais  
✔️ Navegação entre telas  
✔️ Foto carregada via assets  
✔️ Interface moderna com Ionic
